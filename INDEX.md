## 🎯 Common Tasks & Solutions

### Task: Change My Name
1. Open `index.html`
2. Find "Sivesh Kommalapati" (Line 70)
3. Replace with your name
4. Save and refresh browser

**Reference**: [CUSTOMIZATION.md](CUSTOMIZATION.md#-hero-section)

---

### Task: Update Skills
1. Open `index.html`
2. Find Skills section (Line 110)
3. Add/remove `<span class="skill-tag glow">Skill Name</span>`
4. Save and refresh

**Reference**: [CUSTOMIZATION.md](CUSTOMIZATION.md#-content-changes)

---

### Task: Change Color Scheme
1. Open `style.css`
2. Edit root variables (Lines 2-12)
3. Change hex color values
4. Refresh browser to see changes

**Reference**: [CUSTOMIZATION.md](CUSTOMIZATION.md#-color-theme)

---

### Task: Deploy to Web
1. Choose platform (Netlify recommended)
2. Follow [DEPLOYMENT.md](DEPLOYMENT.md)
3. Get live URL
4. Share with everyone!

**Reference**: [DEPLOYMENT.md](DEPLOYMENT.md#-deployment-platforms)

---

### Task: Add My Resume
1. Convert to PDF
2. Save as `resume.pdf` in root folder
3. Done! Button works automatically

**Reference**: [ASSETS_GUIDE.md](ASSETS_GUIDE.md#-adding-resume)

---

### Task: Make It Faster
1. Compress images with [TinyPNG](https://tinypng.com)
2. Minify CSS/JS (optional)
3. Use CDN (automatic on Netlify)
4. Test with [PageSpeed Insights](https://pagespeed.web.dev)

**Reference**: [DEPLOYMENT.md](DEPLOYMENT.md#-performance-optimization)

---

### Task: Fix Mobile View
1. Press F12 in browser
2. Click mobile icon
3. Test different screen sizes
4. Use [CUSTOMIZATION.md](CUSTOMIZATION.md#-responsive-breakpoints) for CSS changes

**Reference**: [QUICKSTART.md](QUICKSTART.md#-testing-before-launch)

## 🐛 Troubleshooting

### Issue: Page looks blank
**Solution**: Make sure you opened `index.html` in a browser, not in text editor
**Read**: [QUICKSTART.md - First Steps](QUICKSTART.md#-first-steps-5-minutes)

---

### Issue: Styles look broken
**Solution**: Clear cache with Ctrl+Shift+Delete, then refresh
**Read**: [QUICKSTART.md - Common Issues](QUICKSTART.md#-common-issues)

---

### Issue: Changes not showing
**Solution**: Use hard refresh (Ctrl+Shift+R) or clear cache
**Read**: [QUICKSTART.md - Common Issues](QUICKSTART.md#-common-issues)

---

### Issue: Mobile view is broken
**Solution**: Press F12, check responsive view, adjust CSS
**Read**: [README.md - Responsive Design](README.md#-responsive-design)

---

### Issue: Can't deploy
**Solution**: Follow [DEPLOYMENT.md](DEPLOYMENT.md) step-by-step
**Read**: [DEPLOYMENT.md - Troubleshooting](DEPLOYMENT.md#-troubleshooting)

---

### Issue: 3D sphere not showing
**Solution**: Check browser compatibility (Chrome recommended)
**Read**: [QUICKSTART.md - Common Issues](QUICKSTART.md#-common-issues)
